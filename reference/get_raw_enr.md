# Download raw enrollment data for Georgia

Downloads enrollment data from the Governor's Office of Student
Achievement (GOSA) downloadable data repository.

## Usage

``` r
get_raw_enr(end_year)
```

## Arguments

- end_year:

  School year end (e.g., 2023-24 = 2024). Valid range: 2011-2024

## Value

List with enrollment data frame(s)
